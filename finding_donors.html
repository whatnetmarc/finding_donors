开始¶

在这个项目中，你将使用1994年美国人口普查收集的数据，选用几个监督学习算法以准确地建模被调查者的收入。然后，你将根据初步结果从中选择出最佳的候选算法，并进一步优化该算法以最好地建模这些数据。你的目标是建立一个能够准确地预测被调查者年收入是否超过50000美元的模型。这种类型的任务会出现在那些依赖于捐款而存在的非营利性组织。了解人群的收入情况可以帮助一个非营利性的机构更好地了解他们要多大的捐赠，或是否他们应该接触这些人。虽然我们很难直接从公开的资源中推断出一个人的一般收入阶层，但是我们可以（也正是我们将要做的）从其他的一些公开的可获得的资源中获得一些特征从而推断出该值。

转换倾斜的连续特征¶

练习：数据预处理¶

从上面的数据探索中的表中，我们可以看到有几个属性的每一条记录都是非数字的。通常情况下，学习算法期望输入是数字的，这要求非数字的特征（称为类别变量）被转换。转换类别变量的一种流行的方法是使用独热编码方案。独热编码为每一个非数字特征的每一个可能的类别创建一个“虚拟”变量。例如，假设someFeature有三个可能的取值A，B或者C，。我们将把这个特征编码成someFeature_A, someFeature_B和someFeature_C.

特征X 特征X_A 特征X_B 特征X_C
B 0 1 0
C ----> 独热编码 ----> 0 0 1
A 1 0 0

评价模型性能¶

在这一部分中，我们将尝试四种不同的算法，并确定哪一个能够最好地建模数据。四种算法包含一个天真的预测器 和三个你选择的监督学习器。

评价方法和朴素的预测器¶

CharityML通过他们的研究人员知道被调查者的年收入大于\$50,000最有可能向他们捐款。因为这个原因*CharityML*对于准确预测谁能够获得\$50,000以上收入尤其有兴趣。这样看起来使用准确率作为评价模型的标准是合适的。另外，把没有收入大于\$50,000的人识别成年收入大于\$50,000对于CharityML来说是有害的，因为他想要找到的是有意愿捐款的用户。这样，我们期望的模型具有准确预测那些能够年收入大于\$50,000的能力比模型去查全这些被调查者更重要。我们能够使用F-beta score作为评价指标，这样能够同时考虑查准率和查全率：

$$ F_{\beta} = (1 + \beta^2) \cdot \frac{precision \cdot recall}{\left( \beta^2 \cdot precision \right) + recall} $$

尤其是，当 $\beta = 0.5$ 的时候更多的强调查准率，这叫做F$_{0.5}$ score （或者为了简单叫做F-score）。

通过查看收入超过和不超过 \$50,000 的人数，我们能发现多数被调查者年收入没有超过 \$50,000。如果我们简单地预测说“这个人的收入没有超过 \$50,000”，我们就可以得到一个 准确率超过 50% 的预测。这样我们甚至不用看数据就能做到一个准确率超过 50%。这样一个预测被称作是天真的。通常对数据使用一个天真的预测器是十分重要的，这样能够帮助建立一个模型表现是否好的基准。 使用下面的代码单元计算天真的预测器的相关性能。将你的计算结果赋值给'accuracy', ‘precision’, ‘recall’ 和 'fscore'，这些值会在后面被使用，请注意这里不能使用scikit-learn，你需要根据公式自己实现相关计算。

回答：这个模型的优势为

 - 算法简单直观，实现比较简单
 - KNN 方法直接利用了数据之间的关系，减少了类别特征选择不当对分类结果对影响
 - KNN方法在类别决策时，仅与距离最近对几个样本有关，因此能够避免样本数量不均衡造成的问题
 - KNN的分类规则就是数据本身，因此不要求数据一致性问题，即允许数据存在噪声

针对多分类问题，或者说分类条件或特真比较不明显对情况下，样本量足够，此模型的表现比较好。

这个模型的缺点是什么？什么条件下它表现很差？

回答：这个模型的缺点包括

- 分类速度慢，KNN是一种懒惰式学习器，对于特征维度为D，样本个数为N的情况，其分类对时间复杂度是O(DN)。
- 当样本数量非常不均衡时，如某一类型样本数量很大，某一类型样本数量过小时，有可能对于新输入节点，K节点大部分都为大数量类型样本，此缺点可通过设置权值来改进。
- 模型分类输出是非规格化的，不是类似概率的输出，分类的可解释性不像决策树那么强，分类输出全取决于数据。

根据我们当前数据集的特点，为什么这个模型适合这个问题。

回答：当前数据集难以根据分类特征对用户进行明显对划分或归类；对输出的时间要求不强，没有速度要求；样本量足够；基于上述几点，合适使用KNN算法。

- 非线性映射是SVM方法的理论基础,SVM利用内积核函数代替向高维空间的非线性映射，因此，SVM 对于特征维度比较高的问题也能很好的解决。

回答： 最终的效果，准确率和F分数都有小幅的下降。如果训练时间是一个考虑因素，我会考虑使用部分特征作为训练集，还有可能会采用更小的数据集来进行训练。但舍弃部分特征实际可能限制了模型能达到的最佳效果，有句话很经典：“数据决定了最终能达到的极限，不同的算法只不过是去逼近那个极限”，因此应对舍弃对特征进行认真分析，不能随意舍弃。
